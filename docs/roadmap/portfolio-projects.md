# Portfolio Project Roadmap

## Flagship Tracks

1. AWS RAG Tutor
2. Research Agent System
3. Local LLM Coding Assistant
4. LLM Evaluation Lab
5. AI Infrastructure Reference Architectures

## Build Order

1. Launch platform base:
   - Homepage, project index, work-with-me funnel
   - Contact API with persistence
2. Deliver one deep demo:
   - RAG tutor with citations and architecture writeup
3. Add agent project:
   - Tool-calling loop and trace logging
4. Add evaluation lab:
   - Model comparisons on latency/cost/quality
5. Publish infra deep dives:
   - Design decisions, failure modes, cost strategy

## Recruiter Signal Checklist

- Clear architecture diagrams
- Public demos with real endpoints
- Cost and scaling tradeoff explanations
- Postmortem-style notes on what failed and changed
- Practical enterprise constraints (security, latency, budget)
